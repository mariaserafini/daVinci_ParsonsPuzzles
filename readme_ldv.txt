Per caricare un puzzle andare nella pagina:
https://codio.github.io/parsons-puzzle-ui/dist/

copiare-incollare il codice e cliccare "generate"
cliccare in alto a sx su "export"
aggiungere prefisso (tipo nome file senza spazi)
cliccare su "copy code"
incollare codice su un nuovo documento
salvare il documento con estensione .md e caricarlo nella cartella parsons
nel file index.markdown copiare-incollare una delle ultime linee di file (link) e creare un nuovo link per il documento appena fatto

poi, fare commit su github (su vscode basta cliccare sul simbolino tipo rete con pallini, il 3 dall'alto e poi sul pulsante "commit e push") 


spiegazione completa di come usare il generatore (con disturbatori, variabili ecc) qui https://codio.github.io/parsons-puzzle-ui/
